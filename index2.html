What is a Software Engineer?

A software engineer is a person who works on the design, maintence, and testing/development of software applications. Considering how our technologies are advancing by the day for things like automobiles, aviation, data management, telecommunications, factory control, robotics, defense, and security. Though, there are 2 common types of software engineers as well. The Applications Developers mainly design computer applications, like games for example, for consumers. They can also make custom software for a specific person or commercial software that can be sold to the general public. Some of them also build databases or programs that can be used internally or online. The Systems Developers make operating systems make operating systems, regardless if for the public or for an organization. These systems keep computers functioning can they control most of consumer electronics in use today, including cell phones and cars. Systems Developers also build the interface that let people interact with the computer. Keep in mind, there are also a lot more types of software engineers, but these are just the 2 most common types of software engineers. But overall, software engineers usually need these skills: knowing coding languages like Python, Java Script, C, C++, they also need to know object oriented programming, database architecture, agile and scrum project management, operating systems, cloud computing, version control, and design testing and debugging.
